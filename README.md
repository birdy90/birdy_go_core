# Birdy Go Core

Built with `Vite` + `React` + `TypeScript SWC` + `Tailwind`. 

Besides that, I'm using:
- `react-aria-components`
- `react-icons`
- `react-hook-form`
- `@tanstack/react-table`
- `nodemailer`

## Environment variables

CHROMATIC_PROJECT_TOKEN - to build and publish *Storybook* to *Chromatic*
